How to Use Admin Portal

Step-by-step guide for managing users

Step 1 — Connect to backend

  1. Go to Settings (bottom of sidebar)
  2. Enter Backend URL: https://jyotish-ai-4xw2.onrender.com
  3. Enter your email and password under "Admin Login"
  4. Click Login & Save Token — this gets your JWT
  5. Click Test Connection to verify it works

Step 2 — Add a new user

  1. Click Add User in the sidebar
  2. Enter Full Name and Email
  3. Set a strong Password (min 8 chars)
  4. Choose Access Level:
    • Free — basic horoscope only
    • Premium — Kundli, AI Chat, Matchmaking, Muhurtham
    • Admin — everything + admin portal access
  5. Set Status to Active and click Create User
  6. Share credentials with the user

Step 3 — Change user access level

  1. Go to All Users
  2. Find the user and click Edit
  3. Change Access Level dropdown:
    • Free → Premium: unlocks all features
    • Premium → Admin: gives portal access
    • Admin → Free: removes all special access
  4. Click Save Changes
  5. User gets new access on next app launch

Step 4 — Send a push notification

  1. Go to Push Notifications
  2. Select target: All / Premium / Free users
  3. Enter Title and Message
  4. Optionally add a deep link (e.g. jyotish://horoscope)
  5. Click Send Notification
  6. Result shows how many users were reached
  7. Note: Users must have opened the app at least once to have an FCM token

Step 5 — Delete a user

  1. Go to All Users
  2. Find the user and click Delete
  3. Confirm in the prompt
  4. User is permanently removed from Firestore
  5. Note: This does not revoke active JWT tokens immediately
